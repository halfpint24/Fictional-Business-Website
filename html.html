HTML Introduction

What is HTML?

HTML(Hyper Text Markup Language) is a language that is used to define the basic structure and content of a web page. It is designed to represent visual content on the screen, in contrast to XML, which defines only abstract data. HTML makes up a part of the "Web Trifecta", which includes HTML, CSS, and JavaScript.

Origin

HTML was first conceived in 1989 by computer scientist Tim-Berners Lee, and the first version was released in 1993. Since then, HTML has been released in 5 main versions, the most recent being HTML5(released 2014).

How does HTML work?

First of all, HTML code must be written in a text editor. It is possible to write it in a simple editor like Notepad or TextEdit, but I recommend Notepad++ for beginners. I do not recommend a professional editor like Adobe Dreamweaver just yet, as it allows one to "cheat" and fill in code, which is not good when first learning a language. In addition to text editors, HTML can be made with a Graphical User Interface(GUI) editor, but it will not be taught in this tutorial.

Once the HTML code has been written, it is now ready to run in a web browser. The purpose of a web browser is to read and display HTML documents. The web browser does this by taking the code and using a rendering engine to convert it into a displayable form.

Your First Web Site

  1. Open Notepad++. Press CTRL + ALT + S to open the save dialogue. Name the file index.html. In the save as type dropdown menu, select Hyper Text Markup Language File. Note: file names can only contain letters, numbers, dash(-), underscore(_), and tilde(~). Note: We named the file index.html because browsers load html pages named index first under a given domain name. If you omit this in a real web site, then browsers will not know where to start and give an error.
  2. Now it is time to write the code. All HTML documents must start with a document declaration so that the browser knows what version of HTML to use. Type <!DOCTYPE html> to do this.
  3. An HTML document is made up of tags. One type of tag is a container tag, which contains a start tag and and end tag. One type of a container tag is the <html> tag, which specifies that the code is HTML. All code minus the doctype must go in here. To make this tag, go under your !doctype and type:
    <html>

    </html>
    Notice that the end tag has a forward slash(/) in front of the tag name, while the opening tag does not. If you omit this, then the browser can not tell where the tag starts and ends, which may result in an undesirable output.
  4. Let's put another tag in here. Using the same format from the html tag(and written inside of), write the <head> tag, complete with an opening and closing tag. It is written the same, but you write <head> instead of <html>
  5. Inside the head tag, we need to specify a character set. This is a set of letters, numbers, and symbols that the browser uses. Everyone agrees that UTF-8 is the best one. To use this character set, we use the meta tag, which is used to describe data about your site. Syntax(way of writing the code) for the meta tag is a little different. It is a self closing tag and has something called an attribute:
    <meta charset="UTF-8" />
    An attribute specifies additional data about a tag. In this case, it is the character set. Also make sure not to use a closing tag like the other ones. Write it like it is written here.
  6. Inside the head tag, write <title>My First Web Site</title>, you can replace My First Web Site with anything you want. This is the text that will appear on the tab on the top of the screen.
  7. We have written some basic HTML code, but our webpage is still just a blank screen. Let's add something right now. Add these tags:
    <body>
    <h1>Anything you want</h1>
    <p>Anything you want</p>
    </body>
    The body tag defines the visual content on the page.
    The p tag defines a paragraph, and by default it has a Times New Roman font and a 16 pixel font size.
    The h1 tag defines a header. By default, it is bolded and has a big font size. There are also other header tags that make the text smaller. These tags are h2, h3, h4, h5, and h6. There is no h7, it only goes to h6.
  8. We should be done! At the top of the screen, click Run and then Launch in Chrome, or whatever browser you want. Alternatively, you can press CTRL + ALT + SHIFT + R to launch in Chrome.
